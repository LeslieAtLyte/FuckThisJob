Time to create a bot on Lyte!

Creating a bot on Lyte should be as easy as pie. Just by pressing a button filling out two fields, you'll have your own custom bot. How do I create a bot?
  • Navigate to the Discord Developer Portal then click on the blue button that says "Create New Application" and give your bot a cute name.
  • Now you've made the application, click on the "Bot" tab on the left-hand side. This is where we'll do most of our configuring for your bot.
  • Here we are going to turn off and on a few options, look and read the image below:
02:13
  • Now that we made sure that these are all correct, we are going to grab our token. The token is what actually powers your bot. Think of the token like an account login. This is also how we turn on your bot. Go ahead and scroll all the way up. You should see this button called "Reset Token". Go ahead and click that. If you have 2FA enabled on your Discord account then you'll need the code. Now you should see a long string of characters and numbers. Go ahead and copy that. Make sure to store this in a document because we will need this later.
    • Now we are going to invite our bot to our server. On the dashboard, above the button thats highlighted blue, you should see OAuth2. Go ahead and select that, when you have selected that 2 more options should have appeared below the tab. Select the tab that says "URL Generator". You need to select a few more options here. I've placed an image below, you can refer to it for the options needed.
02:13
  • Now that you've selected the correct options, scroll all the way down. You should see the Generated URL. On the right-hand side of that URL you should see copy button, click on that button.
    • Create a new tab on your web browser and paste that url inside of the big search bar at the top. Now go ahead and invite the bot to the respected server.
    For the next steps, you need to have Developer Mode. You can enable this by going to Settings>Advanced>Developer Mode.
    • Now navigate to the server which you invited the bot to and right click on the bot and select "Copy User Id"
    Make sure to store this in a document because we will need this. Now we can create our bot. Here in the main Lyte server under this message, there should be an embed with a button on it. Click on the button and paste in Discord Bot Token which we got earlier inside of the first box and the Bot User Id which we just got into the last one. Press submit and now your bot should be created. IF YOU BOT IS NOT ONLINE IN 1 MINUTE CONTACT SUPPORT BY DMING @ModMail
